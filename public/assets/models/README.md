# Mission Mode Models

Place GLB/GLTF files here to replace fallback low-poly meshes.

Expected model names are referenced by `public/data/missions.json`, for example:

- `school-building.glb`
- `sports-field.glb`
- `tree.glb`
- `rescue-tent.glb`
- `rubble.glb`
- `crops.glb`
- `wind-zone.glb`
- `package.glb`
- `delivery-pad.glb`
- `charging-station.glb`
- `shelf.glb`
- `space-base.glb`
- `asteroid.glb`
- `crystal.glb`

If a file is missing or fails to load, mission mode creates a low-poly fallback mesh automatically.
