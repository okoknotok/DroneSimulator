# Mission Mode Audio

Place optional MP3 files here to replace generated Web Audio fallback effects.

Expected filenames are listed in `public/data/assets-manifest.json`, for example:

- `mission-start.mp3`
- `mission-success.mp3`
- `mission-fail.mp3`
- `scan.mp3`
- `pickup.mp3`
- `dropoff.mp3`
- `charge.mp3`
- `warning.mp3`
- `ambient-campus.mp3`
- `ambient-rescue.mp3`
- `ambient-farm.mp3`
- `ambient-warehouse.mp3`
- `ambient-space.mp3`

If files are missing, mission mode uses generated Web Audio beeps/noise so the game still feels complete.
